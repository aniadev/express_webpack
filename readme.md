# Express - Webpack
## _Base app using expressjs and webpack_
## Development:

- Run webpack to build bundle
    ```sh
    yarn install
    yarn webpack
    ```
- Switch new Terminal
     ```sh
    yarn dev
    ```
## Production:

- Install
    ```sh
    yarn install
    ```
- Run app
     ```sh
    yarn start
    ```

## Config .env:
- .env
    ```sh
    NODE_ENV = 'development'
    PORT = 8080
    ```
